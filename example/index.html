notivuecation

Promise-based alert/confirm modal for Vue.js. Results are be logged to the console. Note that the styling and looks are completely up to you.

init

import notivuecation from 'notivuecation';

Vue.use(notivuecation);

basic alert/confirm

Using predefined button labels, you can quickly open a confirm or alert popup by just supplying the message.

this.$confirm('Are you sure you want to do this?');
this.$alert('Please click ok');

custom alert/confirm

Send an object instead of a string to define title, message and button labels.

this.$confirm({
  message: 'Are you sure you want to do this?',
  confirm: 'Yes!',
  cancel: 'Hell no',
});

this.$alert({
  title: 'Important!',
  message: 'Please confirm',
  confirm: 'Sure!',
});

fully custom

If you want fully custom buttons, you can call $notify directly and specify all buttons.

this.$notify({
  title: 'Custom title',
  message: 'Custom message',
  buttons: [
    {label: 'option #1', value: 1},
    {label: 'option #2', value: 2},
    {label: 'option #3', value: 3},
  ]
})
